    # 上下文类

    下表中所列出的上下文类允许您改变表格行或单个单元格的背景颜色

    类 描述
    .active 对某一特定的行或单元格应用悬停颜色

    .warning 表示一个需要注意的警告

    # 定义按钮的样式

    Class 描述
    btn 为按钮添加基本样式
    .btn-default 默认/标准按钮
    btn-primary 提供额外的视觉效果，标识一组按钮中的原始动作。
    btn-success 表示一个成功的或积极的动作。
    btn-info 信息警告消息的上下文按钮。
    btn-warning 表示应谨慎采取的动作。
    btn-danger 表示一个危险的或潜在的负面动作。
    btn-link 并不强调是一个按钮，看起来像一个链接，但同时保持按钮的行为。
    <!-- 标准的按钮 -->
<button type="button" class="btn btn-default">默认按钮</button>
<!-- 提供额外的视觉效果，标识一组按钮中的原始动作 -->
<button type="button" class="btn btn-primary">原始按钮</button>
<!-- 表示一个成功的或积极的动作 -->
<button type="button" class="btn btn-success">成功按钮</button>
<!-- 信息警告消息的上下文按钮 -->
<button type="button" class="btn btn-info">信息按钮</button>
<!-- 表示应谨慎采取的动作 -->
<button type="button" class="btn btn-warning">警告按钮</button>
<!-- 表示一个危险的或潜在的负面动作 -->
<button type="button" class="btn btn-danger">危险按钮</button>
<!-- 并不强调是一个按钮，看起来像一个链接，但同时保持按钮的行为 -->
<button type="button" class="btn btn-link">链接按钮</button>

    # 下拉菜单

    • 所有的下拉菜单，必须放到一个dropdown的类里面

    • 在dropdown 的div里面，嵌套一个按钮(button)，为按钮添加dropdown-toggle类，和自定义样式data-toggle="dropdown"

    • 为按钮添加一个向下的箭头<span class="caret"></span>

    • 在按钮的下面添加一个无序列表(ul...li),然后再无序列表(ul)里添加加dropdown-menu

      <div class="dropdown">
	<button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
        下拉菜单
        <span class="caret"></span>
    </button>
    <ul class="dropdown-menu dropdown-menu-left">
        <li><a href="#">Bootstrap</a></li>
        <li><a href="#">HTML</a></li>
        <li><a href="#">Javascript</a></li>
    </ul>
</div>

    # 下拉菜单标题和分割线

    • 标题：直接在你要添加的标题的li标签上面体添加dropdown-header类
    • 分割线：在你需要添加分割线的地方，添加一个空的li标签，为这个li标签添加一个divider

    # 下拉菜单的禁用

    • 在需要禁用的li上添加一个disabled类
    • 在下拉菜单的禁用时，只是会改变下拉菜单的样式，并不会禁用下拉菜单的功能
