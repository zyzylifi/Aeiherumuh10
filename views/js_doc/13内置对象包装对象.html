# Math对象

它是一个全局对象，是一个系统已经定义好的对象，不需要我们通过new关键字来进行构造。主要的作用是用于我位的数学运算

# 常数

  • E，自然对数的底，约等于2.718.....
  • LN2,2的自然对的值，约等于0.693....
  • LN10，10的自然对的值，约等于2.302....
  • PI，圆周率，约等于3.14
  • SQRT2,2的平方根,约等于1.414...
  • SQRT1_2, 指0.5的平方根,约等于0.707...

上面的值是固定的，不可更改，上面的常数可以看成是属性（只是不可更改）

# 延时调用与循环调用

延时调用也叫一次性定时器，循环调用我们也叫循环定时器。而定时器，我们可以理解为闹钟

延时调用相当于一次性的闹钟，例如，我们定义一个闹钟，这个闹钏在某个时间去执行，但是要注意，这个闹钟只响一次（只执行一次）

    console.log(new Date());

# 循环调用

循环调用也叫循环定时器，它与一次性定时器非常相像，它可以重复的进行延时调用

循环定时器使用的方法是setInterval()它的参数与setTimeout保存一致

  1. 第一个参数代表时间到了以后要做的事情（要调用的方法）handler
  2. 第二个参数代表隔多长时间做一次（也就是延时的时间）timeout
  3. 后面跟要传入到方法里面的参数arguments

语法格式如下：

setInterval(handler,timeout,...arguments);
1

下面的代码就是写了一个循环定时器，每隔一秒钟就调用一次方法a,然后创建Date对象，打印当前时间

//定义一个方法，打印本地时间
function a(){
    var d=new Date();   //创建的对象指向当前时间
    console.log(d.toLocaleString());
}
setInterval(a,1000);

清除定时器

循环定时器的清除与一次性定时器的清除是一样的，它使用clearInterval,它们同样需要一个定时器的编号

当我们去得到这个编号以后，我们就可以把这个定时器清除掉了

当我们现在学了循环调用以后，我们不妨把上面的那个从1~10隔秒打印的练习题再拿出来做一下

//隔秒打印1~10
var i=1;
//循环定时器与一次性定时器一样，在设置的时候都会返回一个ID
var id = setInterval(function(){
    console.log(i);
    i++;
    //当i的值大于10以后由清除这个定时器
    if(i>10){
        clearInterval(id);
    }
},1000);

📘 总结：

  1. 延时调用与循环调用它们都会定时执行，但延时调用无特殊情况只会执行一次，循环调用是每隔一段就自己去做
  2. 它们的参数相同，第一个参数代表时间到了以后要调用的方法，我们一般使用匿名方法做回调，第二个参指延时的时间，后面的参数则代表要传递到回调方法里面的实参
  3. 它们在设置定时器的时候都会返回这个定时器的编号，我们可以通过这个编号去使用clearTimout或clearInterval去清除这个定时器

# 包装对象

通过typeof检测出来的数据类型有以下几种

  1. string字符串类型
  2. number数字类型
  3. boolean布尔类型
  4. undefined未定义类型
  5. object对象类型
  6. function方法类型

在上面这些用来表示类型的关键字里面，它们的首字母都是小写

但是有些情况与，我们发现电脑里面与之有一个同名的首字母大写的英语文单词

string-------String,number-------Number,boolean-----Boolean,object-------Object

概念：什么是包装对象？包装对象指的是String,Number以及Boolean这些对象，这三种包装对象其实本质上面指的是string,number以及boolean这三种基本数据类型

包含对象其实就是为之前的基本数据类型里面的三个类型服务的，因为有了这三个包装对象以后，字符串，数字以及布尔布类型就可以像对象一样去调用它们的属性以及方法了

string.__proto__===String;
number.__proto__===Number;
boolean.__proto__===Boolean;
1
2
3

它具备像上面这种特点以后，那么字符串类型，数字类型以及布尔类型就可以像对象一样去分别调用它们父级里面的方法与属性了
