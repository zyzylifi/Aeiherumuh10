# 面向对象程序设计（高级篇）

面向对象指的是万事万物皆对象，如果我们把所有的东西都看成是对象，那么，之前的基本数据类型呢

在JavaScript里面，我们有5种基本数据类型以及一种复杂数据类型

  • String字符串类型
  • Number数字类型
  • Boolean布尔类型
  • Null空类型
  • Undefined未定义
  • Object复杂数据类型

它们这6种数据类型在内存当中的存储形式是什么样子呢？

# 基本类型的值传递

值传递也叫副本传递（拷贝一个副本传递过去）

var a=123;      //定义了变量a
var b=a;        //把a的值赋值给变量b
a = 456;        //将a的值重新设置为456;
console.log(a);      //456
console.log(b);      //123
//这个时候a与b的值分别是多少

var c="hello";
var d=c;
c="word";
console.log(c);      //world
console.log(d);      //hello
//这个时候a与b的值分别是多少

通过上面的例子，我们可以看到a与b，c与d这些变量之间它们互不影响

要弄清楚上面的特点，我们就不得不弄清楚一个点就是内存存储数据的形式。首要先明白一点，我们所定义的所有变量或方法全部都保存在计算机的内存（RAM）当中，这些数据在内存当中是怎么存放的呢？

在JavaScript里面，数据在内存当中的存放形式主要是“堆”和“栈”，栈里面存储是基本数据类型 ，而堆里面存储的是复杂数据类型

栈的特点：固定大小空间，任何编程语言都可以直接操作栈里面的数据

基本数据类型在做传递的时候使用值传递，两个变量之间互不影响

# 复杂数据类型的引用传递

引用传递也叫地址传递，它传递的是堆里面的内存地址

堆的特点：不固定大小空间，编程语言不能够直接深入到堆里面去操作数据

对象它固定不了大小，所以它不能够直接放在内存的栈里面去

var a={
    userName:"杨标"
};    			//创建一个对象a
var b=a;		//把a赋值给对象b
a.userName="张三";
console.log(a.userName);   //张三
console.log(b.userName);

# 对象的浅拷贝

  1. 浅拷贝这个概念只应该发生在复杂数据类型上面
  2. 浅拷贝只是在栈里面在操作，如果对象是浅拷贝，那么，它只是在栈里面复制一份相同的内存地址，这个时候，它们存放的地址是相同的，所指向的堆里面的对象也是同一个对象

因为对象是复杂数据类型，存放在内存的堆里面，所以，它当它在做传递的时候，它使用是地址传递，这种情况会造成很大的麻烦。如果我们一个变量发生改变以后，势必会影响到指向它的所有变量

var obj={
    userName:"张三",
    sex:"男"
}
var obj1=obj;
//这一种方式的拷贝会有很大的问题，当我们去改变obj的时候，另一个也会发生改变
//obj与obj1其实是指向了同一个内存的堆的对象，只要我改变一个，另一个肯定会发生改变
//上面的var obj1=obj这就是一个浅拷贝的过程

现在，如果我们要复制一个互不影响的对象，这应该怎么办呢

# 对象的深拷贝

  1. 深拷贝的概念也只发生的对象上面
  2. 它与浅拷贝不同的就是，它不仅仅要操作栈里面的数据，还要操作堆里面的数据，拷贝出来的两个对象互不影响
  3. 深拷贝是要深入到堆里面去操作的

各位同学始终记得一点，基本数据类型是做值传递，它们互不影响，所以我们可以通过这个特点来完成我们对象的深拷贝

浅拷贝和深拷贝的分析

var obj={
    age:18,
    sex:"男",
    hobby:["看书","睡觉"]
}
var obj1=obj;   //浅拷贝   改变一个，另一个也改变
/*
    typeof obj=="object"  对象  对象是存在堆里面，把地址放到了栈里面，你拷贝的时候拷贝的是地址
    这个地址指向了同一个对象
    所以一个改变，另一个肯定会改变
*/
var age=obj.age;    //不影响
/*
    typeof obj.age==="number"  基本数据类型   值传递，值传递互不影响
    age=obj.age
*/

var hobby = obj.hobby;  //浅拷贝
/*
    type obj.hobby==="Object"  对象是在堆里面，把地址放在了栈里面，拷贝的时候拷贝的是栈里面地址
    这个地址指向了同一个对象
    所以一个改变，另一个也改变
*/

# 常规深拷贝

# 【真·深拷贝】

我们已经发现系统提供给我们的方法是不能够实现完全的深拷贝的，所以，我们现在在这里手动去实现深拷贝

第一个版本

 /**
     * 我给你一个旧的对象，你还我一个新的对象
     * 我给你一个旧的数组，你还我一人新的数组
*/

function copyObject(oldObj) {
    //要做深拷贝之前，要先判断一下当前这个东西是不是一个对象
    if (typeof oldObj === "object" && oldObj != null) {
        //说明真的是一个对象，我要做深拷贝
        var newObj = {};
        //第一步：先获取所有的属性名
        var props = Object.keys(oldObj); //得到所有的属性名，返回一个包含所有属性名的数组
        props.forEach(function (item, index, a) {
            if (typeof oldObj[item] === "object") {
                //说明是复杂数据类型 
                if (oldObj[item] === null) {
                    //说明是空值
                    newObj[item] = null;
                } else if (Array.isArray(oldObj[item])) {
                    var n = copyArray(oldObj[item]); //我调用封装好的方法，给你一个旧的数组，你还我一个新的数组
                    newObj[item] = n;
                } else {
                    var o = copyObject(oldObj[item]);
                    newObj[item] = o;
                }
            } else {
                //基本的数据类型    function类型
                newObj[item] = oldObj[item];
            }
        });
        return newObj;
    }
    else{
        return oldObj;
    }
}


//我给你一个旧的数组，你还我一个新的数组
function copyArray(oldArr) {
    var newArr = [];
    oldArr.forEach(function (item, index, a) {
        //数组遍历出来的item就代表每一项
        if (typeof item === "object") {
            if (item === null) {
                newArr.push(null);
            } else if (Array.isArray(item)) {
                //如果是一个数组
                var r = copyArray(item);
                newArr.push(r);
            } else {
                //如果它真的是一个对象 又是一个对象的深拷贝
                var ro = copyObject(item);
                newArr.push(ro);
            }
        } else {
            newArr.push(item);
        }
    });

    return newArr;
}

 /**
     * 我给你一个旧的对象，你还我一个新的对象
     * 我给你一个旧的数组，你还我一人新的数组
*/
function copyObject(oldObj) {
    //要做深拷贝之前，要先判断一下当前这个东西是不是一个对象
    if (typeof oldObj === "object" && oldObj != null) {
        //说明真的是一个对象，我要做深拷贝

            if (typeof oldObj[item] === "object") {
                //说明是复杂数据类型 
                if (oldObj[item] === null) {
                    //说明是空值
                    newObj[item] = null;

                    // var n = copyArray(oldObj[item]); //我调用封装好的方法，给你一个旧的数组，你还我一个新的数组
                    var n=  copyObject(oldObj[item]);
                    var newArr=[];
                    Object.assign(newArr,n);

                    newObj[item] = o;

                //基本的数据类型    function类型
                newObj[item] = oldObj[item];
            }
        });
        return newObj;

    else{
        return oldObj;
    }
}
