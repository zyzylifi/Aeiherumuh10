# JavaScript第二课

# JavaScript基本语法

  1. JavaScript是区分大小写的（HTML与CSS 不区分大小写）

  2. JavaScript必须写在<script>标签里面，并且指定正确的类型(type)

    <script type="text/javascript"></script>

  3. JavaScript如果语法出现了错误，那么会在页面上的控制台(console)报错

  4. JavaScript里面，一句代码写完了以后，请加上分号结束;,不加分号理论上是可以的，但是不推荐

# 标识符

用来标明某一些对象或事务的特征，主要应用于关键字与变量名

标识符它一般是指变量名，方法名，关键字或保留字

变量名相当于人的姓名，它可以变，关键字则是系统自已的使用的

if,else,for,while等这些都是系统关键字，而我们用户在写代码的过程当中，自己取的名子我们叫变量名

注意：

  1. 标识符的开头必须是字母、_下划线、$开头，不能以数字开头，后面的则没有要求，可以使用任何这符

    一般我们建议把-换成_下划线

    不要使用中文做变量名，如性别='男';年龄=25等这种方法，虽然这种方法不会报错，但我们不能用

  2. 标识符应该遵循见名知意的原则，JavaScript默认以驼峰命名为原则

userName="yangbiao";  //账号
userNickName="标哥哥"; //昵称
passWord="123456";    //密码

# 变量标识符

什么是变量：变量就是在描述事件特征（属性）当中，可以变化的数据，我们叫变量

使用关键字var来定义的标识符，我们叫变量标识符，变量标识符用于存放一定的数据在代码当中

var userName="杨标";

# 数据类型

当我们定义变量以后，再向变量赋值的过程当中，会存以几种情况的值，例如姓名是汉字，年龄又是数字等

汉字：一二三四五六七八九

数字：123456789

变量后面接的值的数据类型主要有以下几种

  1. 字符串（String）

    指的是汉字，英文等其它的字符

    当定义变量赋值字符串值的时候，我们一定要加上引号 （单引号与双引号都是用来形容字符串）

    加了引号了数字不是数字类型，是字符串类型

  2. 数字（Number）

    指的是0~9之间的数组成的数字（可以是整数，也可以是小数，还有负数）

  3. 布尔类型（Boolean)

    它只有两个值true/false;

  4. 空值（Null）

    当我们在定义一个变量的时候，我们如果不想给它赋值可以直接给一个null的空值

    注意它通过typeof检测出来的结合是object

  5. 未定义（Undefined）

    当一个变量在定义的时候，没有赋值，那么，它就是会出现undefined

上面的五个数据类型是JavaScript当中的基本数据类型

总结：变量有5种基本的数据类型 一种复杂的数据类型（后面会讲object）
