# JS第一课

在Web标准里面，经过与用户或其它的页面产生交互以后所得到的结果，这个过程，我们把它叫作行为。行为是网页当中最重要的一个部分

在网页当中与用户产生交互的试有很多，在以前的时候，我们主要是通过脚本的方式进行，而这些脚本根据不同的浏览器也会有一些细微的差距

主要的脚本有以下几咱：

  1. 以微软为标准的VBScript

    这一种脚本，它以VBS结尾，可以运行在浏览器里面，也可以运行在windows的电脑里面，只是后来被浏览器淘汰了，现在主要运行在windows的电脑时面

    asp+VBScript

  2. 以微软为核心的第二代脚本JSCript

    这一个版本的脚本主言，借鉴了Java的相关语言优势，在网页当中以脚本的形式存在

    J#+JScript

    这一个版本因为Java告微软侵权了，然后被撤销了

  3. Java公司与netscape公司结合开发一款基于页面的脚本主言，它借鉴Java的语言优势，然后只做了小部分的改动，语言命名为JavaScript

    优势：

    • 借鉴了当前非常火的Java语言的面向对象开发的优势，把JavaScript打造了一个基于对象开发的语言、
    • 因为与W3C进行合作，它推出了JavaScript以语言以后W3C就把它设置成了主流的网页脚本语言

目前，我们所使用的JavaScript的版本已经经过了无数次的更新，目前我们所使用的版本是5.1

# JavaScript到底是什么

它是一种脚本语言，提供页面与用户的交互途径，主要包含三个方面的东西

  • ECMAScript(ES)

    它主要是用来定义JavaScript的语法规范，现在主流的版本是5.1,后期的主要部分要放在6.0上面

    【语法，关键字，保留字，类型，操作符，对象，语句】（简单部分）

  • DOM(document object model:文档对象模型)

  • BOM(bowser object model:浏览器对象模型)

# JavaScript运行在什么地方

在设计JavaScript这门语言的时候，我们当初的设计者是要让这一门语言运行在浏览器里面的，所以，我们的JavaScript与CSS一样都是运行在页面当中。但是它也有一个特殊的标签进行包裹

<script type="text/javascript"></script>
1

JS代码与CSS代码有很多的相似之处，主要体现的代码的位置，CSS的代码主要在三个位置，JS代码也就是行内代码，和Script代码，以及我们的外部JS文件

  1. 写在标签里面
<button type="button" onclick="alert('hello world')" style="color:red">
    按钮
</button>

  1. 写在特定的标签里面，与CSS内部样式块相同
<script type="text/javascript">
	alert('你好');
</script>
<style type="text/css">
    div{
        width:100px;

    <!--这是注释-->
