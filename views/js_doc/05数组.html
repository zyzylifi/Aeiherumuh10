# 数组

什么是数组？通过字面的意义直面理解数组就是把数据组合在一起

在C/C++以及Java/C#等语言里面，数组是如下定义一

数组是存放一系列相同的数据类型的组合，这个组合一旦确定长度以后就不可改变

  1. 数据类型相同
  2. 长度固定

JavaScript里面的数定义

数组是存放一系列数据的集合

# 数组的定义

在其它的编程语言里在，创建数组的语法可能稍麻烦 ，但在JS里面，很简单

# 第一种方式Array

这一种方式，我们可以直接使用JS的内置对象Array来进行

# JS数组与其它编程语言数组的异同

# 数组的取值与赋值相同

  1. JS数组的取值与赋值跟其它的编程言是一样的，它们都是通过下标取值与赋值（下标也叫索引，可以把它理解书本上面的页码，通过索引，可以快速的定位到数组里面的值）

  2. JS数组与其它编程语言数组一样，它的索引（下标）是从0开始的，它们的取值与赋值都是数组[索引值]

    var b=["李昊","高自忠","段创创","陈正松","万成"];
b[0];  //取“李昊”
b[1];  //取到“高自忠”
b[2]="张三";   //把值赋到b数组的第3个位置

# 数据类型的区别

数据类型的区别

在其它的编程语言里面，数组里面的值使用的是同一个数组类型，而在JS的数组里面，里面的值的数据类型可以不相同

var arr=["张三",18,true,null,undefined,"",NaN];

数组长度与最大索引的区别

在其它的编程语言里面，数组的长度决定了最大索引，例如如果在C/C++/Java等编程语言里面，如果数组的长度为6，那么它最大的索引应该是5（长度决定最大索引）

在JavaScript里面的数组，最大索引决定了数组当前的长度

结论 :通过上面的相同点与不同点的结论，我们可以得出一个，JS数组不限定长度，也不限制数据类型!(正是因为这个特点，所以JS把数组也当成了集合)

# 数组类型的检测

在平常的工作当中，我们可以通过typeof关键字来检测数据类型，typeof 只能检测了基本的数据类型与对象类型 ，而不能够具体的检测出数组

如果判断这个变量是否是数组，我们还需要做一次检测

# 使用instanceof关键字

instanceof主要的作用与typeof相似，typeof主要用于检测基本数据类型 ，而instanceof主要用于检测对象类型

var num=new Array();
num instanceof Array;   //判断num是否是则Array new出来
1
2

如果 变量 instanceof Array得到的值为true，则代表它是一个数组，否则就不是一个数组

# Array.isArray()检测

除了通过之前的instanceof关键字以外，我们还可以使用自带的方法去检测

var num=new Array();
Array.isArray(num);    //判断num是否是数组
1
2

如果上面的结果为true则说明num 是数组，如果得到false则说明num不是数组

我们也都知道索引是从0开始，到length-1结束，所以我们要在这个范围里面不停的去拿，这个地方我们做了一件重复的事情，就需要傅和循环了

# 归并方法

归并方法也是迭代方法的一种，但也不完全的迭代方法，它的回调方法里面有四个参数

var arr=["a","b","c","d","e"];
arr.reduce(function(prev,current,index,a){
    console.log(prev,current,index);
    return "杨标";
});

  • prev：当执行第一次遍历的时候，它是第一个元素,current才指向第二个元素

    后面则是上一次回调方法返回的值

  • current代表当前的元素

  • index当前的索引

  • a代表当前遍历的数组

案例

var arr=[1,5,7,9,0,3,2,4,8];
//求上面数组中最大的值
var result =  arr.reduce(function(prev,current,index,a){
    return prev>current?prev:current;
});
//求上面数组当中的和
var result2=arr.reduce(function(prev,current,index,a){
    return prev+current;
});

# 排序方法

排序方法sort是数组中常用的排序方法，它可以接收一个函数做为参数，这个函数接收两个参数，如果第一个参数应该位于第二个参数之前，则返回一个负数，如果第一个参数位于第二个参数之后，则返回正数，如果相等则返回0

var arr = [1, 5, 7, 9, 0, 3, 2, 4, 8];
//我现在希望实现一个倒序
arr.sort(function (a,b) {
    if(a>b){
        return -1;
    }
    else if(a<b){
        return 1;
    }
    else {
        return 0;
    }
});
// [9, 8, 7, 5, 4, 3, 2, 1, 0]
