# 二维数组

js在严格的意义上面来说是没有二维数组的概念，它只有多维数组概念

在之前学数组的时候，我们对JS数组的概念定义是存放一系列数据的集合

现在请看下面代码

var name0="张三";
var name1="李四";
var name2="王五";
var name3="赵六";
//现在我们有四人变量，这样写义很麻烦 ，所以我们可以通过数组的形式来定义，集中管理
var names=["张三","李四","王五","赵六"];

# 二维数组的定义

二维数组的定义与一维数组的定义方式相同，它们都有两种方式

  1. 通过Array的方式来定义

    var stus=new Array(2);
stus[0]=new Array(4);
stus[1]=new Array(4);

    var stus=new Array([],[]);   //这是可以的
//-------------------------------
var stus=new Array(new Array(),new Array());   //这是可以的

var stus=new Array(new Array(4),new Array(4));   //这也是可以的

# 二维数组的取值与赋值

在数组里面，我们的取值方式是通过数组名[索引]来取值

var stus=[["张三","李四","王五","赵六"],[18,19,29,25]];
//如果我想取到里面的"李四"，以及29应该怎么做呢？
var names=stus[0];   //取到了stus里面的第一个元素，它是一个数组
//它又是一个数组数，所以我又可以通过数组的方式去取值
var name1=names[1];   //得到了“李四”

二维数组的取值我们划分成了两部分

var name1= stus[0][1];  //"李四"
var age1=stus[1][2];   //29

2

JS的取值与赋值我们可以把它看成是一个表来进行操作

JS为什么会存在二维数组的概念是因为JS如果在内部要实现表的数据结构是通过二维数组来进行的，但是JS又不是一个完整意义上的表

var stus=[["张三","李四","王五","赵六"],[18,19,29]];
1

在上面的数组里面，我们可以看到姓名在这里是四个元素，而年龄则是三个，在其它的编程语言这是不允许的，但是在JS里面，因为JS没有限定数组的长度，所以这是正常的

var stus=[["张三","李四","王五","赵六"],[18,19,29],true,"hello world"];
//这一种定义方式它也可以的
1
2

JS没有严格意义上面的二维数组，它有只有多维

var infos = [["张三","李四","王五","赵六",["杨标","周洋"]],[18,19,29]];
//现在如果要取“杨标”
var names=infos[0][4][0];
