# 响应式布局

一个页面针对不同的设备不同的宽度做出的不同样式布局响应

在整个移动端布局里面，我们都需添加如下的网页头

<meta name="viewport" content="width=device-width,initial-scale=1,user-scalable=no" />
1

那么现在我们就可以根据网页的宽度来进行响应了

# 响应式布局实现原理

它的原理主要是对CSS里面的一个命令来完成，这个命令叫媒体查询

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width,initial-scale=1,user-scalable=no" />
		<title>响应式</title>
		<style type="text/css">
			.div1{
				height: 100px;
				border: 1px solid red;
				color: red;
			}
            
            /*当屏幕设备在0~500px之间的时候，所使用的CSS样式*/
			@media only screen and (max-width:500px) and (min-width:0px){
				.div1{
					color: blue;
				}
			}
		</style>
	</head>
	<body>
		<div class="div1">
			今天天气真热，感觉要下雨了
		</div>
	</body>
</html>
<!--要求：我希望这个网页如果在屏幕大于500px的时候里面的字体就显示为红色 ，否则就显示会蓝色-->

# 屏幕的划分

  • 手机（0px~768px）
  • 平板（769px~990px）
  • PC（991px~1200px)
  • 大型PC（投影仪，高分屏显示器）(1201px以上的)

我们之前在学习font-size的属性的时候，学过一个属性值叫rem, 这个是响应式单位

rem指的是root element网页的根元素html ,而html网页默认的字体大小是16px

默认1rem=16px，我们可以通过一个像这样的特性来完成响应式网页的字体的设置

				border: 1px solid red;

如果一段文字没有限范围的改变，只是需要让它随着设备变大变小，那么，我们可以使用另一种单位，叫vw或vh

vw: viewport width，1vw 代表设备宽度的1%

这一种情况不需要借助于媒体查询，因为它没有限定范围

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width,initial-scale=1,user-scalable=no" />
		<title>字体响应式</title>
		<style type="text/css">
			.div1{
				border: 1px solid red;
				/*如果现在我想设置12px*/
				font-size: 3.2vw;
                /*以iphone6为标准
                12px/375(iphon6的宽度)*100=3.2
                */
			}
		</style>
	</head>
	<body>
		<div class="div1">
			我要随着页面改变字体大小
		</div>
	</body>
</html>

# 不规则图片响应式

当图片大小不一致的时候，这个时候，如果我们在设置height:auto这个时候我们就会出现高度不统一的情况，当高度不统一的时候，我们就很难使用第一种方式做到图片响应式，这样会留下很大的空隙

针对这种情况，我们可以使用第二种方案

多列+媒体查询

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" content="width=device-width,initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no" />
		<title></title>
		<style type="text/css">
			*{
				margin: 0;
				padding: 0;
				list-style-type: none;
			}
			.imgBox{
				column-count: 4;
				column-gap: 0px;
			}
			.imgBox>img{
				width: 100%;
			}
			@media only screen and (max-width:768px){
				.imgBox{
					column-count: 1;
				}
			}
			@media only screen and (min-width: 769px) and (max-width: 990px) {
				.imgBox{
					column-count: 2;
				}
			}
		</style>
	</head>

		<div class="imgBox">
			<img src="img/1.jpg" alt="" />

		</div>
	</body>
</html>

# 自适应布局

让一个页面在不同的设备宽度上面自动去适应布局

现在讨论，怎么样去实现上面的布局，左边的图标大小固定，在不同的设备上面压缩的时候，只压缩右边的空白区域

# 第一种方案，定位

<!DOCTYPE html>
<html lang="zh">
<head>
	<meta charset="UTF-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1.0" />
	<meta http-equiv="X-UA-Compatible" content="ie=edge" />
	<title>自适应</title>
	<style type="text/css">
		*{
			margin: 0;
			padding: 0;
			list-style-type: none;
		}
		.item{
			height: 45px;
			border-bottom: 1px solid gray;
			position: relative;
		}
		.left{
			width: 45px;
			height: 45px;
			background-color: red;
		}
		.right{
			height: 45px;
			background-color: lightblue;
			position: absolute;
			right: 0px;
			top: 0px;
			left: 45px;
		}
	</style>
</head>
<body>
	<div class="box">
		<div class="item">
			<div class="left"></div>
			<div class="right"></div>

</body>
</html>

# 第三种方案，弹性盒子（弹性布局）

在以前讲盒子模型，我们引出了标准文档流，在浮动与定位里面，我引出了脱流（非标准文档流），这种流式布局的方式是以前为了应对PC端的布局而产生的布局规范

现在，我们在移动端下面，更多情况下不建议使用流式布局规范来完成移动端页面的布局，转而使用另一种布局规范，弹性布局

普通的元素遵循的是标准的盒子模型，我们只需要在标准的盒子模型上面添加一个属性，那么，这个盒子就换成了弹性盒子

<!DOCTYPE html>
<html lang="zh">
<head>
	<meta charset="UTF-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1.0" />
	<meta http-equiv="X-UA-Compatible" content="ie=edge" />
	<title>自适应</title>
	<style type="text/css">
		*{
			margin: 0;
			padding: 0;
			list-style-type: none;
		}
		.item{
			height: 45px;
			border-bottom: 1px solid gray;
			display: flex;  /*转换成弹性盒子了*/
			
		}
		.left{
			width: 45px;
			height: 45px;
			background-color: red;
		}
		.right{
			height: 45px;
			background-color: lightblue;
			width: 200px;
			flex: 1;
		}
	</style>
</head>
<body>
	<div class="box">
		<div class="item">
			<div class="left"></div>
			<div class="right"></div>
		</div>
	</div>
</body>
</html>

# 设置全屏DIV大小

现有如下一段代码，请将.app的div设置全屏

<!DOCTYPE html>
<html>
	<head>
		<meta name="viewport" content="width=device-width,initial-scale=1,minimum-scale=1,maximum-scale=1,user-scalable=no" />
		<meta charset="UTF-8">
		<title>弹性布局案例</title>
		<style type="text/css">
			*{
				margin: 0px;
				padding: 0px;
				list-style-type: none;
			}
			
			/*第二种：同时设置body与html的高度为100%;
			 * html,body{
				height: 100%;
				width: 100%;
			}
			.app{
				width: 100%;
				height: 100%;
				background-color: red;
			}*/
			/*直接去掉前面的doctype文档声明部分，采用非标准模式开发（混杂模式）
			 * .app{
				width: 100%;
				height: 100%;
				background-color: red;
			}*/
			.app{
				width: 100vw;
				height: 100vh;
				background-color: deeppink;
			}
		</style>
	</head>
	<body>
		<div class="app">
		</div>
	</body>
</html>
